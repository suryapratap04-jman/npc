# 13. Developer Guide

This document describes instructions for contributing to the repository.

## 1. Coding Guidelines
- Follow standard Python pep8 casing rules.
- Maintain directory structure cleanliness.
- Avoid placing custom scripts outside of the `scripts/` or `experiments/` folders.

## 2. Test Execution
Unit tests are written in Pytest:
```bash
.venv\Scripts\pytest backend/tests/
```
